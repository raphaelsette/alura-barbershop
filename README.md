
# Oracle Next Education + Alura

Desenvolvimento de um site estático para o curso de iniciante em HTML5 e CSS3.

O curso abrange os seguintes tópicos:

- Listas e Divisões
- Navegação
- Posicionamento de elementos
- Pseudo-classes CSS
- Formulários
- Semântica
- Estrutura de tabelas
- Design responsivo
