
# Oracle Next Education + Alura

Development of a static website for the beginner course in HTML5 and CSS3.

The course covers the following topics:

- Lists and Divisions
- Navigation
- Element positioning
- CSS Pseudo-classes
- Forms
- Semantics
- Table Structure
- Responsive Design
